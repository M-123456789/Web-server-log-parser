**Цель задания:** Разработать консольное приложение на Python, которое анализирует лог файл и выводит статистику по этим логам.

**Описание:**

Лог файл имитирует записи о работе веб-сервера. Каждая строка лога содержит дату и время события, IP-адрес пользователя, тип запроса (GET, POST и т.д.), URL, код ответа сервера и размер ответа в байтах. Ваша задача — написать скрипт, который анализирует этот файл и выводит следующую статистику:

1. Общее количество запросов.
2. Количество запросов по типу (сколько GET, POST и т.д.).
3. Топ 5 самых часто встречающихся IP-адресов.
4. Топ 5 самых больших запросов по размеру ответа.

**Требования к выполнению:**

- Использовать ООП: создать класс для анализа логов с соответствующими методами.
- Код должен быть читаемым и хорошо структурированным.
- Обеспечить обработку исключений для предотвращения сбоев при чтении файла или некорректных данных.
- Предоставить пример лог файла для тестирования скрипта.
- Написать документацию к коду (комментарии к методам, описание работы скрипта).

**Формат лога для примера:**

```
2024-03-01 12:14:15,192.168.1.1,GET,/home,200,512
2024-03-01 12:14:16,192.168.1.2,POST,/api/data,200,1024
...
```
